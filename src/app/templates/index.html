Metadata form

Please enter the package metadata (top-level metadata) in the form below. Mandatory fields are indicated with an asterisk *.

Add one author per author field. Use the [+] button to add extra authors. The same for keywords.

The field "type of research" is mandatory, since it determines the retention period.

When you are finished filling out the form, use the [Download] button to save the metadata as a file metadata.json.

Please enter all required fields! Title is required. Author name is required. Description is required. Keyword term is required. Related datasets is required.
